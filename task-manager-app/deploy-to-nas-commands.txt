SSH COMMANDS TO RUN ON NAS (copy-paste these one at a time):

# 1. Navigate to app directory
cd ~/molino-app

# 2. Load the new Docker image
docker load -i molino-task-manager-latest.tar

# 3. Stop and remove the old container
docker-compose down

# 4. Start the new container
docker-compose up -d

# 5. Wait 3 seconds and check logs
sleep 3

# 6. View the server logs (Ctrl+C to exit)
docker logs molino-briganti-task-manager
